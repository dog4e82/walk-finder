# 우리 학교 산책지수 🚶

공공데이터포털의 **기상청 단기예보 API**와 **에어코리아 대기오염정보 API**를
결합해 우리 학교 주변의 **시간대별 산책 적합도 점수**를 계산하는 웹앱입니다.

> 산책 약속을 잡거나 "지금 나가도 될까?" 고민될 때 한눈에 보기 위해 만들었어요.

## 🌟 차별점
- **두 개의 공공데이터 API를 결합**해 단순 조회가 아닌 *새로운 지수*를 산출
- **시간대별 점수**로 "오늘 몇 시가 가장 좋은가"까지 추천
- 학생들이 일과 중 활용 가능한 6–21시 범위로 추천 시간 한정
- 토스 스타일의 깔끔한 카드 UI + 친구 같은 추천 메시지

## 🧰 기술 스택
- **백엔드**: Python 3 (urllib, json, math 표준 라이브러리만 사용 → 외부 패키지 0개)
- **프론트엔드**: 정적 HTML / CSS / Vanilla JS
- **데이터**: 공공데이터포털 OpenAPI 2종

## 📡 사용 API
| API | 용도 | 데이터셋 ID |
|---|---|---|
| 기상청 단기예보 조회서비스 | 시간대별 기온·강수확률·풍속·하늘상태 | `15084084` |
| 한국환경공단 에어코리아 대기오염정보 | 학교 인근 PM10·PM2.5 | `15073861` |

## 🧮 산책지수 계산 공식
**총점 = 기상 점수(0–50) + 대기 점수(0–50)** 로 0–100점.

### 기상 점수 (만점 50)
- 기온: 15–22℃ 만점, 범위 벗어날수록 감점
- 강수확률·강수형태: 비/눈일 때 -30, 강수확률 60% 이상이면 -18
- 풍속: 9 m/s 이상 -12, 5 m/s 이상 -5
- 하늘상태: 흐림 -3

### 대기 점수 (만점 50)
환경부 등급 기준으로 PM10·PM2.5 중 더 나쁜 등급을 채택:
- 좋음 50 / 보통 38 / 나쁨 18 / 매우나쁨 5

## 🚀 로컬 실행

### 1. API 키 설정
1. `.env.example` 을 `.env` 로 복사
2. `.env` 안의 `SERVICE_KEY` 자리에 공공데이터포털에서 발급받은
   **일반 인증키(Decoding 버전)** 을 붙여넣기
   - `.env` 는 `.gitignore` 에 의해 GitHub 에 올라가지 않습니다

### 2. 데이터 받아오기
```bash
py fetch_data.py
```
실행하면 `data.json` 이 생성/갱신됩니다.

### 3. 웹 페이지 열기
정적 사이트이므로 간단한 로컬 서버로 띄우면 됩니다:
```bash
py -m http.server 8000
```
그 후 브라우저에서 `http://localhost:8000` 접속.
